jenkins 构建时选择分支
发表于2022-08-08|Jenkins部署
安装Git ParameterJenkins选择分支需要安装 Git Parameter 插件 配置 安装完毕之后在打开当前工程的配置选项 然后找到General，选中This project is parameterized 点击添加参数按钮，选择 Git Parameter 输入名称变量为branch，输入描述，参数类型选择分支或标签，默认值可以填写自己默认的分支 然后切换到源码管理选项，在指定分支处填入刚刚的名称变量 $branch，然后保存即可。 启动，选择分支最后返回到工程，点击build with Paramenters, 就可以选择项目的分支部署啦
